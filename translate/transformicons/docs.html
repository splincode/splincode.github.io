Как выглядит стартовый макет

<!DOCTYPE html>
<html>
 <head>
  <meta charset="utf-8">
  <link rel="stylesheet" href="css/tcon.css">
  <script src="js/tcon.js"></script>
 </head>
 <body>
  <!-- Тут используем код наших кнопок -->
 </body>
</html>

Настраиваем внешний вид

Для того, чтобы было удобно управлять кнопками, изменять их размеры и цвет, необходимо подключить плагин tcon.set.js на ваш сайт после основного скрипта.

Параметры javascript-объекта tcon

Метод Входные параметры Установка параметров кнопки
set (string, {object}) Установка ширины, высоты и фона
