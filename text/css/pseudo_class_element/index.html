Псевдо-элементы
Псевдо-классы
- Существующие DOM-элементы, отвечающие
определенному состоянию
- Перед именем псевдо-класса ставится двоеточие
- Основные псевдо-классы:
:hover
:first-child
:nth-child()
:not()
Псевдо-элементы
- Отвечают за определенную часть DOM-элемента
- Перед именем псевдо-элемента ставится два
двоеточия
- Основные псевдо-элементы:
::before
::after
::first-later
::first-line
::selection
Псевдо-элементы
- Отвечают за определенную часть DOM-элемента
- Перед именем псевдо-элемента ставится два
двоеточия
- Основные псевдо-элементы:
::before
::after
::first-later
::first-line
::selection
::before
- добавляет контент перед содержимым внутри
DOM-элемента
- добавленный контент наследует стиль от
элемента, к которому он добавляется
-использование в css-файле
DOM-элемент::before { content: "текст“; }
::after
- добавляет контент после содержимого внутри
DOM-элемента
- добавленный контент наследует стиль от
элемента, к которому он добавляется
-использование в css-файле
DOM-элемент::after { content: "текст“; }
::after и ::before
- домашнее задание
::first-letter и ::first-line
::first-letter определяет стиль первого символа в
тексте
::first-line задает стиль первой строки
форматированного текста
-использование в css-файле
DOM-элемент::first-letter { color: red; }
DOM-элемент::first-line { color: red; }
::selection
- применяет стиль к выделенному пользователем
тексту
-использование в css-файле
DOM-элемент::selection { ... }
